# 🎓 學術產出自動對應系統 | Academic Data Automation

### 📖 專案背景 (Context)
本專案為模擬過去的實務經驗。
單位需要定期統計校內教授的學術產出（論文發表、Impact Factor影響指數），以進行學術評鑑。面臨以下問題：

1.  **資料源格式不一**：內部人事系統匯出的姓名常包含**多餘空白**或全形半形差異，導致無法直接與外部資料庫比對。
2.  **人工比對耗時**：傳統上會使用 Excel VLOOKUP 函數，需人工逐筆檢查資料，當資料量越來越多時，人工方式效率相對較低且出錯率相對較高。

---

### 💡 解決方案 (Solution)
使用 **Python 自動化清洗與比對腳本**，取代人工行政作業。
核心邏輯包含：
1.  **自動清洗 (Data Cleaning)**：利用 `String Manipulation` 技術，批量移除資料中的格式雜訊。
2.  **智慧串接 (Smart Merging)**：使用 `Pandas Merge (Left Join)` 邏輯，精準將內部名單與外部論文數據整合。
3.  **績效計算 (Aggregation)**：自動計算每位教授的總學術積分，並產出排序後的報表。

---

### 🚀 專案成果 (Impact)
* **準確率提升**：提升資料正確性：透過程式邏輯解決 Excel 資料格式與隱藏字元問題，完全消除人為肉眼比對之失誤風險。
* **效率優化**：將原本每月結算需耗時 約 2-3 個工作天 的人工比對工作，透過自動化腳本縮短至 5 分鐘內 完成（含資料匯入與報表產出）。
* **可複製性**：模組化與擴充性：此自動化邏輯具備高可移植性，已建立通用模組，可快速應用於部門內其他 「跨系統帳務勾稽」 流程。

---

### 🛠️ 核心技術 (Tech Stack)
* **Language**: Python 3
* **Library**: `Pandas` (核心數據處理)
* **Key Techniques**:
    * `.str.strip()` (資料清洗/去空白)
    * `pd.merge(how='left')` (資料庫關聯/串接)
    * `.groupby().sum()` (數據統計)
    * `.fillna()` (缺失值處理)
